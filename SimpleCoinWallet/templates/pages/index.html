Introduction

This is A Simple Web Application Design To Perform Very Simple Transection Handling Between Users And Miners. Actually, The Interesting Point About this Application is, This Application is Based On Blockchain concept. In Simple Words, To Understand And Test My Blockchain And Bitcoin Technology Knowledge, I tried to write a Simple Web Application That Can Use A Json File As A Blockchain Storage Database To Record All The Transection, And Sqlite3 To Store Users Authentication Data. Note: Blockchain Json File is Public Available To Every One. I Tried To Implement Some Protection Mechanizms To Stop our Imaginery Attacker Like Double Spending Detection Mechanizm, Proof Of Concept, Signature Verification, Transection Verification And much more.

SimpleCoin

As I already mentioned in above text, SimpleCoin is a Simple BlockChain Based Ledger Implemented To handle Transections between Users. This Application can easily perform money transefer between two user and can also protect and detect, if any malicious editing is performed in Blockchain Json File. I tried my best to make this project easy to understand and easy to use but still, i think its little hard to understand for new programmers those who are not aware of cryptograpy Technology and bitcoin Technology. Actually, Main reason for the complexity of this Implementation is Bitcoin/Blockchain concept because In my personal opinion, Bitcoin and blockchain is little hard for newbies to Understand and Also hard to Implement in Any program.
